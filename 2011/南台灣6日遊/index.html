day4 社頂公園生態之旅 -> 草地球 -> 南灣水上活動 -> 後壁湖輝哥生魚片(午餐) ->白砂沙灘夕陽漫步 -> 白砂音樂酒吧 -> 恆春市區 -> 墾丁大街(晚餐) -> 伊朵民宿

day5 九棚港口飆沙 -> 茂伯的家 -> 照利河豚料理(午餐) -> 海生館 -> 高雄六合夜市(晚餐) ->85大樓民宿
